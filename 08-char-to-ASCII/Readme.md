# Program to convert character into ASCII value in C

```c
#include <stdio.h>
int main()
{
    printf("Enter character : ");
    char c;
    scanf("%c", &c);
    printf("ASCII code is %d", c);
    return 0;
}
```
### Terminal Output #1
```
Enter character : j
ASCII code is 106
```
### Terminal Output #2
```
Enter character : A
ASCII code is 65
```
