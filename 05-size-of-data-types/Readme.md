# Program to find area of a circle in C

```c
#include <stdio.h>
int main()
{
    int x;
    float f;
    char c;
    double d;
    printf("Size of int is : %d\n", sizeof(x));
    printf("Size of float is : %d\n", sizeof(f));
    printf("Size of char is : %d\n", sizeof(c));
    printf("Size of double is : %d\n", sizeof(d));
    return 0;
}
```
