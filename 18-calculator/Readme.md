# Program to find area of a circle in C

```c
#include <stdio.h>
int main()
{
    float radius;
    printf("Enter the radius : ");
    scanf("%f", &radius);
    float area = 3.14 * radius * radius;
    printf("Area of circle is %f", area);
    return 0;
}
```

```

Area of circle is 9847.040039
```
